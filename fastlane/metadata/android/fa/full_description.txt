Hermes2 اولین پورت نیتیو اندرویدِ Hermes Agent است — ایجنت هوش مصنوعی متن‌باز از Nous Research. به‌جای تایپ‌کردن دستور در ترمینال، یک اتاق فرمان درست‌وحسابی با Material 3 برای ایجنت خودت داری که کاملاً روی گوشی اجرا می‌شود.

ایجنت کار را انجام می‌دهد — گفت‌وگو، اجرای ابزار، نوشتن کد. Hermes2 یک خانه‌ی راحت روی اندروید برایش می‌سازد.

قابلیت‌ها

• استریم زنده‌ی پاسخ‌ها، توکن‌به‌توکن
• نمایش استدلال — فکرکردن مدل را قبل از پاسخ ببین
• کارت‌های فراخوانی ابزار با خروجی جمع‌شونده و کارت‌های زیرایجنت
• دستورهای اسلش مستقیم از کاتالوگ gateway
• کشوی سشن — جست‌وجو، مرتب‌سازی، سنجاق، تغییرنام، حذف و ادامه
• تأیید ابزار به‌صورت اعلان اندروید (Approve / Deny)
• صفحه‌های تنظیمات، مهارت‌ها، Cron و Platforms
• ۶ تم رنگی با حالت‌های روشن / تاریک / سیستم
• رابط دوزبانه — انگلیسی و فارسی

حریم خصوصی

• ارتباط اپ ↔ ایجنت روی 127.0.0.1 (loopback) است و هرگز دستگاه را ترک نمی‌کند.
• بدون حساب کاربری، بدون تله‌متری، بدون واسطه‌ی ابری.
• کلید API تو داخل تنظیمات خود Hermes در Termux می‌ماند.
• پیام‌هایت فقط به ارائه‌دهنده‌ی مدلی که انتخاب می‌کنی می‌رود.

نیازمندی‌ها

این یک ابزار حرفه‌ای است، نه اپ چت معمولی. به Termux (از F-Droid) با نصب Hermes Agent نیاز دارد، به‌علاوه یک کلید API برای ارائه‌دهنده‌ی مدل مثل Xiaomi MiMo یا Google Gemini. ایجنت می‌تواند دستورهای واقعی شل را داخل Termux اجرا کند — تأیید ابزار را روشن نگه دار و گوشی‌ات را روت نکن.

پروژه‌ی مستقل و جامعه‌محور. وابسته به Nous Research نیست و مورد تأیید آن‌ها نیست.
